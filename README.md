# How to Init

Windows:

`cd WebApp`

`sqlite3 -init BanCobradotas.sql BanCobradotas.db`

Unix:

`$ cd WebApp`

`$ sqlite3 -init BanCobradotas.sql BanCobradotas.db`

# How to Test

`$ cd Test/`

`$ dotnet test`
